# game-pages

## Structure

- `index.html`: 首页入口页
- `pages/castle-defense.html`: 游戏二级页
- `styles/home.css`: 首页样式
- `styles/game.css`: 游戏页样式
- `scripts/game/main.js`: 游戏逻辑脚本

## Static Deployment

This project is pure static files and can be deployed to any static hosting:

- GitHub Pages
- Vercel (static)
- Netlify
- Nginx static directory

## Local Preview

You can open files with a static server, for example:

```powershell
cd d:\git_repo\game-pages
python -m http.server 8080
```

Then open:

- http://localhost:8080/
